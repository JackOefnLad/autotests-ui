9. Обнаруженные баги

Баг №1: Кнопка Registration не проверяет заполненность поля Username
Описание: В коде кнопки условие disabled проверяет ""===u.email||""===u.password||""===u.password (password проверяется дважды, username не проверяется). Таким образом, кнопка может быть активна при пустом Username.
Ожидаемое поведение: disabled: ""===u.email||""===u.username||""===u.password
